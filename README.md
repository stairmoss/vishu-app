# vishu-app


This is a simple web application built using Next.js for sending and sharing Vishu wishes. The goal of this app is to make it easy for users to generate, customize, and share festive greetings with friends and family.

## About the Project

Vishu is an important festival in Kerala, and people usually send wishes to their loved ones. This app provides a clean and easy way to create and share those wishes online.

The app is built with a focus on speed, simplicity, and a smooth user experience.

## Features

* Generate Vishu wishes instantly
* Clean and simple user interface
* Mobile-friendly design
* Easy sharing options

## Tech Stack

* Next.js
* React
* JavaScript
* CSS

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

Make sure you have installed:

* Node.js
* npm or yarn

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/vishu-wishes-app.git
```

Go to the project folder:

```bash
cd vishu-wishes-app
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and go to:

```
http://localhost:3000
```

## Project Structure

```
/pages        -> Application routes
/components   -> Reusable UI components
/styles       -> CSS files
/public       -> Static assets
```

## Usage

Open the app, generate a Vishu wish, and share it with your friends or family. The app is designed to be simple so anyone can use it without confusion.
